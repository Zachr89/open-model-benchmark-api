# open-model-benchmark-api

Production-ready REST API that benchmarks open-weight models (Qwen, Llama, Mistral, DeepSeek) against proprietary ones (Claude, GPT-4) on real-world coding tasks. Developers submit test cases via REST endpoint or GitHub Action, get comparative JSON reports with accuracy/cost/latency scores. Built for indie devs choosing models for agent/tool projects—think Lighthouse CI but for AI model selection.
